# vomaste.cz

Static site, served via GitHub Pages. No build step — Tailwind, Flowbite, and Alpine.js are loaded from CDN directly in each HTML page.

## Structure

- `index.html` — landing page
- `dossier/` — layout template for an investigative-report-style page (placeholder content only)

## Stack

- [Tailwind CSS](https://tailwindcss.com/) (Play CDN)
- [Flowbite](https://flowbite.com/) components
- [Alpine.js](https://alpinejs.dev/) for lightweight interactivity
